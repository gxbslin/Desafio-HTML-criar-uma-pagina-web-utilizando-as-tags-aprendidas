O curso foi apresentado pelo professor Diogo e demonstrou como seria a metodologia de ensino mais focada na prática, nesse segundo momento.

Ferramentas Utilizadas

Para criar páginas web, utilizamos um editor de código para escrever o texto e um navegador para visualizar o resultado. O editor ajuda na organização, enquanto o navegador interpreta as instruções e as exibe graficamente. São os instrumentos básicos de qualquer desenvolvedor.

Inspetor de Elementos

O Inspetor é uma ferramenta embutida nos navegadores que permite visualizar o código-fonte de qualquer site em tempo real. Ele funciona como um "raio-x", permitindo testar alterações e entender como cada parte da página foi construída.

Estrutura Básica do HTML

Todo arquivo HTML segue uma organização padrão que começa com a declaração do tipo de documento. Ele é dividido em um cabeçalho para informações técnicas e um corpo onde fica o conteúdo visível. Essa hierarquia garante que o navegador entenda a página corretamente.

Falando sobre Tags

Tags são as etiquetas que definem o propósito de cada elemento na página, como títulos ou parágrafos. Elas geralmente vêm em pares, com uma abertura e um fechamento . É através delas que damos significado ao conteúdo puro.

Atributos das Tags

Atributos fornecem informações extras sobre uma tag, como o destino de um link ou o nome de uma classe. Eles ficam sempre dentro da tag de abertura e funcionam como propriedades configuráveis. Sem eles, as tags seriam genéricas e menos funcionais.

Textos

A exibição de textos é feita através de tags específicas para títulos, parágrafos e destaques. O HTML organiza a hierarquia visual do conteúdo, diferenciando o que é um cabeçalho principal de um texto comum. Isso facilita tanto a leitura humana quanto a de robôs de busca.

Listas Ordenadas e Não Ordenadas

As listas organizam informações de forma sequencial (com números) ou apenas por tópicos (com pontos). Elas são essenciais para agrupar itens relacionados de maneira clara e legível. O código define se a ordem importa ou se é apenas uma coleção de itens.

Links

Links são a base da navegação na internet, conectando uma página a outra ou a endereços externos. Eles utilizam a tag de âncora e dependem de um atributo de referência para saber para onde levar o usuário. É o que transforma documentos isolados em uma rede conectada.

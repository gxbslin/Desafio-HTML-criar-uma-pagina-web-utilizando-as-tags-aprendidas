Bloco Linguagens de Programação

O que é

São uma maneira de se comunicar com as máquinas, traduzindo a linguagem da máquina(binário) a algo que seja mais compreensivel para nós. O que o cliente vê é o Front-End, executado por lingugaens como JavaScript, que é particularmente uma linguagens que é tanto Client-side quanto Server-side.

Server-Side

É o lado do servidor, onde tudo acontece, onde o programa está, onde a aplicação entende o que deve fazer, armazena, envia, realiza quaisquer metodos predefinidos anteriormente para cada ação.

Client-Side

O cliente, nesse caso, é o que executa os programas, dá o input, que é levado até o servidor e retorna para o cliente o que foi pedido ou executado, a saída, o resultado, o output daquilo que foi feito. É onde linguagens server-side são implementadas, isso é, back-end, como C#, Java, Python, C++, entre muitas outras.

HTML não é Linguagem de Programação

HTML não se encaixa como linguagem de programação porque nao pode se comunicar e nem trazer comandos diretamente para a máquina, é apenas uma ferramenta visual, que existe somente dentro de um browser(navegador), por isso carregar o nome de Hipertexto.
